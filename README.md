Capsli – Medikamentenverwaltung
«Capsli» ist eine einfache React-Native-App, mit der Sie Ihre Medikamente verwalten können. Sie erfassen Einnahmezeiten, Notizen und Bilder zu jedem Medikament und behalten so den Überblick über Ihre Behandlung.

Funktionen
Dashboard mit Begrüssung und Hinweis bei leerer Liste
Liste aller erfassten Medikamente
Detail- und Bearbeitungsformular:
Name
Dauer (permanent oder von–bis)
Einnahmezeiten
Notiz
Bild
Einstellungen (z. B. Benachrichtigungen, Sprache)
